# MCFishing  
Minecraft Automatic Fishing  
Need use fishingResourcepack  
Support multiple game versions  
  
我的世界自动钓鱼软件  
需要使用钓鱼材质包  
支持多游戏版本  


使用方法:  
加载钓鱼专用材质包  
将游戏窗口化  
打开钓鱼软件  
将钓鱼软件移至一边  
调节X、Y将游戏指针显示在画布中间(稍微偏上  
点击钓鱼软件的启动按钮  
切换到游戏 视角保持在水面  
4s后开始自动抛竿  
视角抬头即可停止钓鱼
